===========================
How to use 1-my_list
===========================

This is a doctest for MyList class.

>>> MyList = __import__('1-my_list').MyList

Test with a basic unsorted list:
>>> my_list = MyList([1, 4, 2, 3, 5])
>>> my_list.print_sorted()
[1, 2, 3, 4, 5]

Test with an empty list:
>>> my_list = MyList([])
>>> my_list.print_sorted()
[]

Test with a list already sorted:
>>> my_list = MyList([1, 2, 3, 4, 5])
>>> my_list.print_sorted()
[1, 2, 3, 4, 5]

Test with a reverse sorted list:
>>> my_list = MyList([5, 4, 3, 2, 1])
>>> my_list.print_sorted()
[1, 2, 3, 4, 5]

Test with a single element:
>>> my_list = MyList([1])
>>> my_list.print_sorted()
[1]

Test with negative numbers:
>>> my_list = MyList([-1, -5, 0, 3, 2])
>>> my_list.print_sorted()
[-5, -1, 0, 2, 3]

Test with duplicate values:
>>> my_list = MyList([1, 1, 2, 2, 3])
>>> my_list.print_sorted()
[1, 1, 2, 2, 3]
