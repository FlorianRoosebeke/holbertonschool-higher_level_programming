This module defines a function ``add_integer(a, b=98)``.

Usage
=====

``add_integer()`` returns the sum of its two arguments.

Importing the function:
    >>> add_integer = __import__('0-add_integer').add_integer

Basic addition with two integers:
    >>> add_integer(1, 2)
    3

    >>> add_integer(100, -2)
    98

Using default value for b:
    >>> add_integer(2)
    100

Addition with floats (converted to int):
    >>> add_integer(100.3, -2)
    98

    >>> add_integer(2.5, 3.5)
    5

Negative numbers:
    >>> add_integer(-5, -5)
    -10

    >>> add_integer(-100, 100)
    0

Large numbers:
    >>> add_integer(1000000, 2000000)
    3000000
