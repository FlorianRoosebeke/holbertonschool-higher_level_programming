# python-object_relational_mapping

## Objectif
Relier Python a MySQL en SQL brut puis via SQLAlchemy ORM.

## Fichiers
- `0-select_states.py`: Script Python: 0-select states.
- `1-filter_states.py`: Script Python: 1-filter states.
- `10-model_state_my_get.py`: Script Python: 10-model state my get.
- `11-model_state_insert.py`: Script Python: 11-model state insert.
- `12-model_state_update_id_2.py`: Script Python: 12-model state update id 2.
- `13-model_state_delete_a.py`: Script Python: 13-model state delete a.
- `2-my_filter_states.py`: Script Python: 2-my filter states.
- `3-my_safe_filter_states.py`: Script Python: 3-my safe filter states.
- `4-cities_by_state.py`: Script Python: 4-cities by state.
- `5-filter_cities.py`: Script Python: 5-filter cities.
- `7-model_state_fetch_all.py`: Script Python: 7-model state fetch all.
- `8-model_state_fetch_first.py`: Script Python: 8-model state fetch first.
- `9-model_state_filter_a.py`: Script Python: 9-model state filter a.
- `model_state.py`: Modele ORM SQLAlchemy representant l'entite State.
- `README.md`: Documentation du depot et des exercices.

## Competences apprises
- Interagir avec MySQL depuis Python
- Ecrire des requetes securisees
