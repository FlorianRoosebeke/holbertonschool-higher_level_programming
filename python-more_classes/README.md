Python - More Classes and Objects

This project is part of the Holberton School Higher Level Programming curriculum.
It focuses on advanced usage of classes and objects in Python, including:
- Private and public attributes
- Property decorators
- Special methods (__str__, __repr__, __del__)
- Class and static methods
- Instance counting
- Custom string representation

## Project Structure

- 0-rectangle.py : Empty Rectangle class
- 1-rectangle.py : Rectangle with width and height, getters/setters
- 2-rectangle.py : Add area and perimeter methods
- 3-rectangle.py : Add __str__ method for string representation
- 4-rectangle.py : Add __repr__ method
- 5-rectangle.py : Add __del__ method
- 6-rectangle.py : Use property for area/perimeter
- 7-rectangle.py : Add class attribute for instance count
- 8-rectangle.py : Add static and class methods
- 9-rectangle.py : Add comparison method and usage example

## Usage

All files are executable and can be run directly with Python 3.8+ on Ubuntu 20.04 LTS.

	$ ./1-rectangle.py
